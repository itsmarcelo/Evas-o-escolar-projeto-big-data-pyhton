# Evas-o-escolar-projeto-big-data-pyhton
Educação em Dados: Análise Estatística da Evasão Escolar Pós-Pandemia na região Nordeste.

# Como rodar projeto

Identificar Arquivos de Dados Necessários: Liste o seguinte arquivo CSV que é essencial para o notebook: 'df_combined_nordeste.csv' (contém os microdados combinados para o Nordeste).

Estruturar Pastas do Projeto: Crie a seguinte estrutura de pastas no seu diretório principal do projeto (onde o notebook estará): uma pasta chamada 'Dados Filtrados para Nordeste'.

Posicionar Arquivos nas Pastas Corretas: Coloque o arquivo 'df_combined_nordeste.csv' dentro da pasta 'Dados Filtrados para Nordeste'. O arquivo 'df_nordeste_final_2019_2023.csv' será gerado pelo próprio notebook na pasta raiz após a execução das células iniciais.

Executar o Notebook Sequencialmente: Abra o notebook e execute todas as células em ordem, de cima para baixo. O notebook possui verificações internas para carregar os DataFrames necessários caso não estejam na memória.
